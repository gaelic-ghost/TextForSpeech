# TextForSpeech v0.20.0

## What changed

- moved URL normalization onto semantic runs backed by `NSDataDetector` link detection
- added semantic-run token routing for file paths and file-line references
- added semantic-run token routing for function calls, issue references, CLI flags, and identifier token families
- preserved replacement-rule priority ordering while migrating token families onto semantic runs
- kept email addresses marked as semantic runs without rewriting them as links
- removed the Gale-specific `galew` and `galem` built-in semantic alias rules
- updated roadmap and maintainer docs for the current semantic-run migration state

## Breaking changes

- none

## Migration notes

- callers that use built-in profiles no longer receive the `galew` or `galem` standalone word aliases by default
- callers that still want those personal aliases should add custom whole-token replacements in their own profile
- URL, path, file-reference, style-sensitive developer token, and identifier normalization behavior remains available through the existing normalization APIs
